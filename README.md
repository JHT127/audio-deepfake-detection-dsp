# Audio Deepfake Detection Using Classical DSP

Detecting AI-generated / spoofed speech using classical digital signal processing techniques вЂ” FFT, spectral feature extraction, filter design, windowing, autocorrelation, Z-transform analysis вЂ” without relying on black-box deep learning.

Built on the ASVspoof2019 LA evaluation dataset. Reproduces and extends the feature pipeline from Alzantot et al. (Interspeech 2019).

**Course:** Digital Signal Processing  
**Partners:** Joud Thaher, Rand Saleh, Rahaf Alqam  
**Result:** 88.3% accuracy В· F1 = 0.885 В· ROC-AUC = 0.938 (SVM, RBF kernel)

## Reproducing the Dataset

The raw audio is not in the repo (~30 MB, ASVspoof2019 license). Run these once from the repo root **in order**:

```bash
python scripts/list_remote_contents.py   # verify remote archive is reachable
python scripts/fetch_protocol.py         # download eval protocol (~2 MB)
python scripts/select_subset.py          # pick 150 real + 150 fake filenames
python scripts/extract_subset.py         # stream only those files (~15-30 MB)
python scripts/verify_subset.py          # confirm all 300 files are present
```

Then preprocess:

```bash
python scripts/process_dataset.py        # .flac в†’ .npy frames (data/processed/)
python scripts/make_split.py             # regenerate data/split.csv if needed
```

Verify everything is correct:

```bash
python scripts/test_handoff.py           # must print: ALL CHECKS PASSED
```

## Source Modules

### `src/preprocessing.py`

Entry point for going from a raw audio file to frequency-domain features.

```python
from src.preprocessing import load_and_frame, get_spectrogram, get_log_spectrogram, SR

frames   = load_and_frame("data/raw/real/LA_E_1027220.flac")  # (num_frames, 400)
spec     = get_spectrogram(frames)                             # (num_frames, 201)
log_spec = get_log_spectrogram(frames)                         # (num_frames, 201)
```

Constants: `SR = 16000`, frame = 25 ms @ 16kHz = 400 samples, 50% overlap, Hamming window.

### `src/features.py`

Extracts a 14-dimensional DSP feature vector per file (mean + std of 7 features):

| Feature | Description |
|---|---|
| Spectral centroid | Weighted mean frequency |
| Spectral bandwidth | Spread around centroid |
| Spectral rolloff | Frequency below which 85% of energy falls |
| Spectral flatness | Tonal vs. noise-like signal ratio |
| Zero-crossing rate | Sign changes per frame |
| Short-time energy | Frame-level energy |
| Autocorrelation | Max normalized lag-1+ correlation |

### `src/classify.py`

Trains an RBF-kernel SVM on the scaled feature matrix and returns accuracy, precision, recall, F1, ROC-AUC, confusion matrix, and a saved confusion matrix plot.

### `src/filters.py`

FIR/IIR filter design and analysis utilities (frequency response, pole-zero plots, group delay).
